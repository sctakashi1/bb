Откройте helper_bot.py, введите свой id телеграма и токен бота


Для запуска на Windows запустите файл Bot_start.bat


1.Скачиваем и устанавливаем Python последней версии (https://www.python.org/downloads/)(Поставьте галочку Add PATH в установщике)
После перезагружаем ПК или дедик
2.Открываем командную строку (cmd)
3.Пишем следующие команды (поочерёдно):
pip install pyTelegramBotApi (ждём установки)
pip install Pillow (ждём установки)
4.Открываем в консоли папку с ботом (cd *ваш путь*)
5.Пишем python helper_bot.py

Бот запущен :) Чтобы выключить - Ctrl + C
